Our Course Offerings

Comprehensive Computer Science Curriculum

We offer a comprehensive range of courses designed to build strong foundational knowledge and advanced expertise in computer science.

Core Courses

Introduction to Programming

Available: Fall, Spring

Fundamentals of programming using Python. Topics include variables, data types, control flow, functions, and basic data structures.

Data Structures & Algorithms

Available: Spring, Summer

In-depth study of essential data structures and algorithmic problem-solving techniques. Includes arrays, linked lists, trees, and sorting algorithms.

Database Management Systems

Available: Fall, Spring

Design and implementation of relational databases. SQL queries, normalization, transaction management, and database security.

Web Development

Available: Spring, Summer

Full-stack web development including HTML, CSS, JavaScript, and backend frameworks. Covers responsive design and modern web practices.

Advanced Courses

Artificial Intelligence

Available: Fall

Fundamentals of AI including search algorithms, machine learning basics, neural networks, and natural language processing.

Cybersecurity Fundamentals

Available: Spring

Network security, cryptography, ethical hacking, and security protocols. Hands-on labs and real-world security scenarios.

Cloud Computing & DevOps

Available: Summer

Cloud platforms, containerization, CI/CD pipelines, and infrastructure as code. Practical experience with AWS and Docker.

Software Engineering

Available: Fall, Spring

Software development lifecycle, design patterns, testing methodologies, and project management in large-scale applications.

Elective Courses

Mobile App Development

Available: Spring, Summer

Building native and cross-platform mobile applications. Covers Android, iOS, and Flutter development frameworks.

Game Development

Available: Fall

Game design principles and implementation using modern game engines. 2D and 3D game development techniques.

Big Data Analytics

Available: Spring

Data processing and analysis with distributed systems. Tools and techniques for handling large-scale datasets.

Internet of Things (IoT)

Available: Fall, Summer

IoT system design, sensor networks, and embedded systems. Practical projects with microcontrollers and sensors.

Course Registration Information

Prerequisites

Most core courses have no prerequisites. Advanced courses require completion of foundational courses. Check the course syllabus for specific requirements.

Credit Hours

Most courses are 3 credit hours. Lab courses may be 4 credit hours. Please refer to the academic calendar for exact credit information.

Class Schedule

Classes are offered in morning and evening sessions to accommodate different student schedules. Online and hybrid options available.
